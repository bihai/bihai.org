1、打开bash
2、安装geany

apt-get install geany*

3、安装Xming
https://sourceforge.net/projects/xming/
4、安装x11

apt-get install libX11-dev libXext-dev libXtst-dev libXrender-dev libxmu-dev libxmuu-dev
 msttcorefonts gtk2-engines-pixbuf```

$ geany

7、安装xterm

apt-get install xterm
